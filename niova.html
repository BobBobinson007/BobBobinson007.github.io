Dein Geist, organisiert.

NotioOS ist ein minimalistisches, Apple-inspiriertes Notizsystem, das Ordnung in deine Gedanken bringt – mit Fokus auf Design und Schnelligkeit.

Einordnungs-Vorschau

Features, die begeistern.

Inbox System

Alles beginnt hier. Jede Idee, jede Aufgabe landet zuerst in der Inbox, damit du dich auf das Wesentliche konzentrieren kannst.

Einordnungsmaschine

Sortiere deine Notizen mit intuitiven Wischgesten. Links für's Archiv, rechts für Workspaces und neu: **Runter zum Löschen**.

Workspaces

Strukturiere dein Wissen in hierarchischen Workspaces. Behalte den Überblick über Projekte, Reflektionen oder Logs.

AI-Suche

Finde Antworten in deinen eigenen Notizen. Unsere KI-Integration versteht deine Daten und hilft dir, Wissen zu reaktivieren.

Privacy First

Deine Daten gehören dir. Mit lokaler Speicherung und 2FA (TOTP & E-Mail) sind deine Notizen sicher verwahrt.

Dark & Light Mode

Wähle zwischen verschiedenen Themen wie Forest oder Sunset. Das System passt sich deinem Workflow an.

Einfach installiert.

$ git clone https://github.com/BobBinson007/oopOS.git

$ cd oopOS

$ npm install

$ npm start

Fertig! Dein System ist unter localhost:3000 erreichbar.
